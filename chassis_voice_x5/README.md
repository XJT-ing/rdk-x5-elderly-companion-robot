# Chassis Voice X5 — 底盘控制与语音交互子系统

本目录为独居老人陪护机器人项目中的底盘语音子系统，运行在其中一块 RDK X5 上，主要负责 R550 麦克纳姆轮底盘运动控制、语音识别、语音播报、大模型任务理解、激光雷达导航与行人跟随等功能。

## 1. 子系统定位

`chassis_voice_x5` 是整机系统中的“移动与交互中心”，主要功能包括：

- 接收老人语音指令；
- 将语音识别结果转换为文本；
- 调用本地或在线大模型进行意图理解；
- 根据语义结果生成底盘运动、导航或任务协同指令；
- 通过 TTS 进行语音反馈；
- 控制 R550 麦克纳姆轮底盘完成移动、转向、停止等动作；
- 结合激光雷达实现建图、定位、导航和避障；
- 通过 ROS 2 DDS 与视觉机械臂子系统通信。
